Introduction

timeToPlay is an icon typeface that turns fractions into countdown/progress clocks for games, like this:

The typeface includes clocks with 1, 2, 3, 4, 6, 8, and 12 sectors. The digits and slash look like this until you complete the fraction:

Note: The colon is only used for one way of creating Powered by the Apocalypse-style clocks. All other icons are made using the slash.

Warning: These icons may not display correctly if the character spacing and/or kerning of the timeToPlay text is set above the default values.

License

timeToPlay is free to use in both personal and commercial projects without paying royalties.

Fonts

timeToPlay comes in various different styles. Aside from Basic (seen above), there are 9 more:

Blot

3/8

Styled as if it was drawn with with a ballpoint or marker pen (an even line width no matter what direction the line's drawn in).

Callig

3/8

Styled as if it was drawn with a calligraphic pen (heavy contrast between strokes in different directions).

Flat

3/8

Sectors have bevelled corners.

Mech

3/8

Sectors have bevelled corners and spokes end in gear teeth.

Pin

3/8

Spokes end in sharp points.

Polygon

3/8

Sectors have completely flat (non-curved/circular) sides.

Round

3/8

Sectors have rounded corners.

Sharp

3/8

Spokes end in sharp arcs.

Thorn

3/8

Spokes end in sharp thorns with curved edges.

Colophon

Type

The heading typeface is Comfortaa v2.3 by Johan Aakerlund and the body typeface is Overpass v3.0 by Delve Fonts. Both are used under the terms of the SIL OFL-1.1.

The fonts can be found in base64 form in the raw HTML of this file (between the <style> tags near the top). You can access the raw code by opening the file in an IDE or a basic text editor, or by viewing the source in a web browser (which may be accessible in your browser by right-clicking the page and selecting an option like “View Page Source”). All fonts in this file are heavily subsetted (subsetting is the process of removing unused glyphs) and may not be suitable for your own projects.

Software

The HTML file was written with BBEdit v14.

The font files were edited with Glyphs v3. The SVG diagrams and the cover image were made with a combination of Affinity Designer v1 and Inkscape v1.

All SVG images were processed using the SVG Optimizer web app SVGOMG and the font and .png favicon were base64-encoded (allowing them to be written directly into this file) using Browserling's file-to-base64 tool.
